# External Inspirations

This harness is informed by external systems, but does not copy them literally.

## Referenced Projects

- `tanweai/pua`: high-agency execution, anti-give-up pressure, verification discipline, and
  proactive extension behavior
- `thedotmack/claude-mem`: persistent memory capture, session continuity, progressive retrieval, and
  durable context storage

## What We Adopt

From high-agency systems:

- push past the first failed attempt
- search before asking
- verify before claiming completion
- continue toward milestone closure

From memory systems:

- store important state outside the chat
- make memory available at session start
- preserve continuity across sessions
- support structured retrieval and handoff

## What We Change For Codex

- we prefer transparent Git-backed memory over opaque hidden memory
- we make roles, inspections, and repairs first-class governance objects
- we use repository artifacts as the baseline sync mechanism for team collaboration
