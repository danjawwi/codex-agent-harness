# Harness Mechanisms

This directory defines the always-on behavior that should activate when the harness is in control.

These are not user-invoked skills.
They are default operating mechanisms embedded into the harness.

The current mechanism set is:

- `search-before-ask.md`
- `high-agency-execution.md`
- `git-backed-memory.md`
- `external-inspirations.md`

Together they provide:

- proactive search before escalation
- stronger execution persistence and verification discipline
- durable memory that survives sessions and can be synchronized through GitHub
